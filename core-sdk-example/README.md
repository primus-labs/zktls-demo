# core sdk example
The Primus core sdk demo. You can run it with the following command.

## Run

```bash
npm install
node index.js
```

## Configuration

Before starting the attestation, you need to configure the following parameters in `index.js`:

### App ID and App Secret
The **App ID** and **App Secret** in `index.js` (lines 6-7) can be obtained from:
- [Primus Developer Hub - My Projects](https://dev.primuslabs.xyz/myDevelopment/myProjects)
- Create a new project or use an existing one to get your App ID and App Secret
